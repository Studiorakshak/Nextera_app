# Nexterra - Professional Hygiene Solutions

This is a modern, fully responsive React web application for Nexterra, built with Vite, React, TypeScript, and Tailwind CSS.

## Features
- Fully responsive modern design (Mobile & Desktop)
- Dynamic product catalog with filtering and search functionality
- Fast page loads using Vite
- Dark mode support

## Getting Started

Follow these steps to run the website locally on your computer:

### 1. Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your computer.

### 2. Installation
Open your terminal in the project directory and run the following command to install the required dependencies:

```bash
npm install
```

### 3. Run the Development Server
Once the installation is complete, start the local development server:

```bash
npm run dev
```

### 4. View the Website
The terminal will provide a local URL (usually `http://localhost:5174/`). Open this URL in your web browser to view the website.

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React (Icons)
- React Router DOM
